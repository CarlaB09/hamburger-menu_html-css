# Hamburger Menu HTML/CSS

A small responsive website demo showcasing a hamburger-style navigation menu built with HTML and CSS.

*  [Live Site URL](https://carlab09.github.io/hamburger-menu_html-css/)

## Overview

This project includes:
- A responsive navigation bar
- A mobile-friendly hamburger menu toggle using a checkbox-based CSS technique
- Font Awesome icons for menu controls and login icon
- Clean styling with gradients and hover effects

## Files

- `index.html` - Main page markup
- `src/css/style.css` - Layout and responsive menu styling
- `src/img/cat-solid.svg` - Logo image used in the navigation bar

## Usage

1. Open `index.html` in your browser.
2. Resize the browser window to see the responsive menu behavior.
3. On smaller screens, click the hamburger icon to open the menu and the close icon to hide it.

## Notes

- The project uses the Font Awesome 6 CDN for icons.
- No JavaScript is required; the menu open/close interaction is handled with CSS and the checkbox input.
